# Arbeiten mit Vignetten-, Fensterbehang- und Möbeldateien{#working-with-vignette-window-covering-and-cabinet-files}

Benutzer des Dynamic Media Classic Image Authoring-Tools können *Vignetten (VNT-Dateien), Fensterbeläge (VNW-Dateien) und Cabinets (VNC-Dateien) verwenden.* Diese Dateien können nicht in Dynamic Media Classic erstellt oder bearbeitet werden.

Bei Vignetten-, Fensterbehang- und Möbeldateien handelt es sich um durch Authoring erstellte Bilder. Mit einem anderen Dynamic Media Classic-Programm namens Image Authoring maskieren Sie Objekte in einem Bild, damit Benutzer dasselbe Bild in verschiedenen Texturen oder Farben sehen können. Zum Beispiel sehen die Benutzer ein mit verschiedenen Stoffen bestücktes Sofa, ein Fenster mit verschiedenen Fensterbehandlungen oder einen Schrank mit verschiedenen Holzarten.

Vignetten-, Fensterbehang- und Möbeldateien werden ebenso wie alle anderen Dateien hochgeladen.
